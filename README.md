# BlockHack – HackVerse 2025 Website
A dynamic, Minecraft-themed website for the HackVerse 2025 hackathon organised by GDG.

## 📌 Overview
BlockHack is a responsive, dynamic website designed for the HackVerse 2025 hackathon.  
It showcases event details, tracks, timeline, and registration in a visually engaging Minecraft-inspired theme.

## 🚀 Features

- Minecraft-inspired UI design
- Responsive layout for all devices
- Event timeline section
- Hackathon tracks and themes
- Registration section
- Smooth scrolling and interactive elements

## 🛠 Tech Stack

- HTML5  
- CSS3  
- JavaScript  
- Bootstrap
